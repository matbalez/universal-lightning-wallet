# Universal Lightning Wallet

Draft specifications for Nostr and Lightning wallet interoperability.

These NIPs specify how applications can integrate third-party Lightning wallets and build Bitcoin payment features using modern Lightning features such as BOLT12.

## Drafts

- [NIP-100: Public BOLT12 Receive Discovery](NIP-100.md)
- [NIP-101: Universal Lightning Wallet](NIP-101.md)
- [NIP-102: BOLT12 Zaps](NIP-102.md)

These documents are drafts for review by Lightning wallet developers, Nostr client developers, and protocol reviewers.
